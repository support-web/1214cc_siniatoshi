# シニアのための投資診断サイト

簡単な質問に答えるだけで、自分に合った投資スタイルやリスク許容度を診断できるWebアプリケーションです。

## 🎯 プロジェクト概要

50代以上のシニア層向けに、投資に対する不安を軽減し、自分に合った投資スタイルを知るきっかけを提供します。

### 主な機能

- ✅ 8つの質問による投資診断
- ✅ 5つの投資タイプに分類（安定重視/堅実/バランス/成長/積極）
- ✅ シニア向けの見やすいUI（大きな文字、明確なコントラスト）
- ✅ LINE公式アカウントへの相談誘導
- ✅ レスポンシブデザイン（スマートフォン最適化）

## 🛠 技術スタック

- **React 18** - UIフレームワーク
- **Vite** - ビルドツール
- **Tailwind CSS** - スタイリング
- **JavaScript (ES6+)**

## 📦 セットアップ

### 前提条件

- Node.js 16以上
- npm または yarn

### インストール

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev

# 本番ビルド
npm run build

# ビルドのプレビュー
npm run preview
```

## 🚀 使い方

1. トップ画面で「診断を始める」をクリック
2. 8つの質問に順番に答える
3. 診断結果を確認
4. LINE公式アカウントで専門家に相談（任意）

## 📱 画面構成

- **トップ画面**: サービス紹介と診断開始
- **質問画面**: 8つの質問（年齢、投資経験、資金、期間、リスク許容度など）
- **診断中画面**: ローディングアニメーション
- **結果画面**: 投資タイプ表示、おすすめ投資先、LINE誘導CTA

## 🎨 デザインガイドライン

シニア層向けのアクセシビリティを重視したデザイン：

- 文字サイズ: 18px以上
- ボタンサイズ: 60px以上の高さ
- 色のコントラスト: WCAG AA基準準拠
- タップ領域: 48px × 48px以上

## ⚙️ カスタマイズ

### LINE公式アカウントURLの設定

`src/components/ResultScreen.jsx` の以下の部分を変更してください：

```javascript
const LINE_URL = "https://line.me/R/ti/p/@xxxxx"; // ← ここを実際のURLに変更
```

### 質問内容の変更

`src/data/questions.js` で質問内容とスコアを変更できます。

### 結果タイプの変更

`src/data/results.js` で投資タイプの内容やスコア範囲を変更できます。

## 📊 診断ロジック

- 各質問の回答に1〜4点のスコアを付与
- 合計スコア（8〜32点）に基づいて5つのタイプに分類
  - 🛡️ 安定重視タイプ: 8〜13点
  - 🌱 堅実タイプ: 14〜18点
  - ⚖️ バランスタイプ: 19〜24点
  - 📈 成長タイプ: 25〜28点
  - 🚀 積極タイプ: 29〜32点
